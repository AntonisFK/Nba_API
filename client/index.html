Currently only serving 2014-2015 season stats

About

Usage

The NBA Players logs Api responds with json to get request.

Get /logs/{:id}

Search with id

/logs/570461629be563e32961cbd0

Get /players/{player name}

When spelling player name use a space and Capitalize first and last name 

/players/ A.J. Price

Get /team/{team}

Capitalize and use abbreviated name to search for team

/team/GSW

Get /player/{player name}/{include}

Search by player and only include certain stats

/player/Klay Thompson/Opp, TOV, PTS, Line

Get /players/{players names}

Get json by players use a space and comma

/players/Klay Thompson, Draymond Green

Get /players/{players names}/{include}

Get json by players and include certain stats. To differentiate players include Player stat

/players/Klay Thompson, Draymond Green/Player, Opp, BLK
